# SaaS Visibility Guard

A Python project for exporting SaaS usage and risk scores as a CSV file.

## Usage

1. Create a `SaaSVisibilityGuard` instance with a list of `SaaSUsage` objects.
2. Call the `export_csv` method to generate a CSV string.
3. Use the `get_saas_usage` method to retrieve a list of `SaaSUsage` objects.

## Testing

Run the tests using `pytest`.
